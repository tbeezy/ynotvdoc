Welcome to ynoTV! A modern, feature-rich FOSS IPTV player built with Tauri and React.

## Installation

Download the latest release from GitHub:

[Download from GitHub Releases](https://github.com/tbeezy/ynotv/releases)

### Windows Installation

- `ynoTV_x.x.x_x64-setup.exe` - NSIS installer (recommended)
- `ynoTV_x.x.x_x64_en-US.msi` - MSI installer

### Install and Run

1. Run the downloaded installer
2. Follow the installation prompts
3. Launch ynoTV

## First Run Setup

Click the **cogwheel icon** on the top right to open Settings.

## Adding Your First Source

After configuring settings, add your IPTV source.

<Tabs>
<TabItem label="M3U Playlist">

1. In Settings, go to **Sources** tab
2. Click **+ Add Source**
3. Select **M3U Playlist**
4. Enter a name and playlist URL (or import from file)
5. Click **Add Source**
6. Click **Sync Channels**

</TabItem>
<TabItem label="Xtream Codes">

1. In Settings, go to **Sources** tab
2. Click **+ Add Source**
3. Select **Xtream Codes**
4. Enter Host URL, Username, and Password
5. Click **Add Source**
6. Click **Sync Channels**

</TabItem>
<TabItem label="Stalker Portal">

1. In Settings, go to **Sources** tab
2. Click **+ Add Source**
3. Select **Stalker Portal**
4. Enter Portal URL and MAC Address
5. Click **Add Source**
6. Click **Sync Channels**

</TabItem>
</Tabs>
