Efficiency is at the core of the ynoTV experience. Master these keyboard shortcuts to navigate the interface and control playback with ease.

<Aside type="tip">
All shortcuts are fully customizable. Navigate to **Settings** > **Shortcuts** to rebind any action to your preferred key.
</Aside>

## Playback Controls

| Action | Default Key | Description |
|--------|-------------|-------------|
| **Play / Pause** | `Space` | Toggle playback state |
| **Mute / Unmute** | `M` | Toggle audio mute |
| **Seek Forward** | `→` | Seek forward in current stream |
| **Seek Backward** | `←` | Seek backward in current stream |
| **Toggle Fullscreen**| `F` | Enter or exit fullscreen mode |

## Navigation & Discovery

| Action | Default Key | Description |
|--------|-------------|-------------|
| **Channel Up** | `↑` | Switch to previous channel |
| **Channel Down** | `↓` | Switch to next channel |
| **Focus Search** | `S` | Focus the search input in the title bar |
| **Close / Back** | `Esc` | Close current view or return to previous screen |

## Interface & Views

| Action | Default Key | Description |
|--------|-------------|-------------|
| **Toggle Live TV** | `L` | Open/Close Live TV view (Guide + Categories) |
| **Toggle Guide** | `G` | Show/Hide the Electronic Program Guide (EPG) |
| **Toggle Categories**| `C` | Show/Hide the categories sidebar |
| **Toggle TV Calendar**| `T` | Open/Close the TV Calendar |
| **Toggle DVR** | `R` | Open/Close the DVR dashboard |
| **Toggle Sports** | `U` | Open/Close the Sports Hub |
| **Toggle Settings** | `,` | Open/Close the global Settings panel |
| **Toggle Stats** | `I` | Show/Hide the playback statistics overlay |

## Layout Modes

| Action | Default Key | Description |
|--------|-------------|-------------|
| **Main View** | `1` | Transition to single main channel view |
| **Picture in Picture**| `2` | Large main window with small secondary overlay |
| **Big + Bottom Bar** | `3` | Large main view with horizontal channel bar |
| **2×2 Grid** | `4` | Four-channel multiview grid |

## Media Tracks

| Action | Default Key | Description |
|--------|-------------|-------------|
| **Audio Menu** | `A` | Open audio track selection modal |
| **Subtitle Menu** | `J` | Open subtitle track selection modal |
