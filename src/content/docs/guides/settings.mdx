## Themes

ynoTV includes 40+ built-in themes:

### Categories

- **Classic**: Dark, Light
- **Glassmorphism**: Ocean, Neon, Galaxy, Autumn, Berry, Forest, Sunset, Rose, Midnight, Amber, Mint, Coral, Lavender, Slate, Cherry, Gold
- **Neon**: Miami Vice, Electric Blue, Hot Pink, Lime, Orange, Red, Yellow, Violet
- **Gradients**: Aurora, Tropicana, Nebula, Monochrome, Neon Nights, Horizon, Dragon Fruit, Arctic, Volcano, Zen Garden, Galaxy, Cyberpunk, Rainbow Prism

Change themes in **Settings** > **Theme**.

## Keyboard Shortcuts

All keyboard shortcuts are customizable:

1. Go to **Settings** > **Shortcuts**
2. Click on a shortcut
3. Press your new key combination
4. Save

### Default Shortcuts

| Key | Action |
|-----|--------|
| `Space` | Play/Pause |
| `M` | Mute |
| `F` | Fullscreen |
| `G` | Guide |
| `C` | Categories |
| `L` | Live TV |
| `R` | DVR |
| `U` | Sports Hub |
| `T` | TV Calendar |
| `S` | Search |

## Import/Export

### Backup Settings

1. Go to **Settings** > **Import/Export**
2. Click **Export Settings**
3. Save the JSON file

### Restore Settings

1. Go to **Settings** > **Import/Export**
2. Click **Import Settings**
3. Select your backup JSON file
4. Confirm import

### What's Included

- All sources and credentials
- Favorites and custom groups
- Category preferences
- Theme and UI settings
- Keyboard shortcuts
